# StudyGram Mobile

スマホで毎日使うことを優先した、一問一答PWAです。

## 機能
- スマホ専用寄りUI
- お洒落なグラデーション/ガラス風デザイン
- 答え入力
- 自動判定
- 答え表示
- 4段階評価
- 忘却曲線風復習
- 問題追加
- 問題編集/削除
- お気に入り
- 苦手モード
- 成績
- JSONバックアップ
- PWA対応

## GitHub Pages
ZIPを解凍して、この中身をリポジトリ直下にアップロードしてください。

必要ファイル：
- index.html
- style.css
- script.js
- manifest.webmanifest
- sw.js
- icon.svg
- README.md
